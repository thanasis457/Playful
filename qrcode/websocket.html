Often, wifi restrictions may not allow you to connect. You can set up a proxy server to tunnel all requests and allow your connection to work over the internet.
Do the following:

  1. Go to ngrok
  2. Sign in/Create account
  3. Make an Domain
  4. Paste the Domain's URL and Authtoken below. Leave empty if you want to use the default local wifi connection
  5. Press submit. Now disable and re-enable the app connectivity on the Menu Bar.
